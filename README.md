# curiosity-ctf (beta)

A hands-on CTF challenge with multiple layers and one final flag.

No walkthrough, no spoilers.

If you like puzzles that force you to look at logs, binaries, traffic, and odd file behavior,
this challenge is for you.

---

## Quick Start

This project is designed for Linux environments (Kali recommended).

Install what you need, then run:

```bash
python3 build_challenge.py
```

That command generates the challenge files in:

```text
challenge/files/
```

---

## What You Get

You receive a small set of files and one encrypted target.

Everything required to recover the final flag exists inside the challenge material itself.

Flag format:

```text
FLAG{...}
```

---
